Parameters

Python

Python runs on the reticulate package.

Jupyter engine uses Papermill style parameters meaning that the cell is tagged “parameters”. Parameters are first class objects - In the example below we can access alpha as an object in python.

#! tags: [parameters]

alpha=0.1
beta=0.1

R

YAML - Yet Another Markup Language

knitr engine - Assign values in YAML. Parameters are then saved in a list called params

They can be accessed with:

  • params$alpha
  • params$beta

Render with parameters:

With the Command Line Interface:

quarto render report.qmd -P alpha:0.2 -P beta:0.2 --output report.html

Note: Parameters must be set in the document before passing them to the render command.

quarto render report.qmd --to html quarto render report.qmd --to pdf
